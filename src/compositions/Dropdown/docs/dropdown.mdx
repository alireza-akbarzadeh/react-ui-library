## Filterable dropdown

<Playground>
  <ThemeWrapper>
    <Flex direction="row" justifyContent="space-between" full wrap>
      <Looper
        list={variants}
        render={variant => (
          <div style={{ height: 300, width: '30%', maxWidth: 300, minWidth: 200, marginRight: 20 }}>
            <InlineDropdown
              iconAfter={<TagIcon size={12} />}
              initialSelected={options[3]}
              inputVariant={variant}
              label="Super awesome dropdown"
              onChange={value => console.log(value)}
              onFilter={handleFilter}
              onSelect={value => console.log('select', value)}
              onSubmit={(event, value) =>
                !options.some(option => option.id === value) && options.push({ id: value, label: value })
              }
              options={options}
              transitional
            />
          </div>
        )}
      />
    </Flex>
  </ThemeWrapper>
</Playground>

## Searchable dropdown

A searchable dropdown is similar to a filterable dropdown except that it starts with no initial values.

<Playground>
  <ThemeWrapper>
    <StateWrapper initialState={''}>
      {({ state: filter, setState: setFilter }) => (
        <div style={{ maxWidth: 300 }}>
          <InlineDropdown
            emptyNotification={filter ? `No options were found for "${filter}".` : ''}
            label="Super awesome dropdown"
            onChange={setFilter}
            onFilter={handleFilter}
            onSelect={value => console.log('select', value)}
            onSubmit={(event, value) =>
              !options.some(option => option.id === value) && options.push({ id: value, label: value })
            }
            transitional
          />
        </div>
      )}
    </StateWrapper>
  </ThemeWrapper>
</Playground>
