The pagination is a wrapper for the [ButtonGroup](../components/ButtonGroup). It can use either `ButtonGroup` styles or custom styles on top of an unstyled `ButtonGroup`.

## Pagination with custom styles

### Page links with icons

<Playground>
  <ThemeWrapper>
    <StateWrapper initialState={8}>
      {({ state: currentPage, setState: setCurrentPage }) => (
        <Pagination
          withIcons

          withPreviousAndNext
          activeProps={{ weight: 'solid', color: 'primary', shape: 'brick', noPadding: true, style: { width: 32 } }}
          inactiveProps={{ weight: 'shaded', color: 'primary', shape: 'brick', noPadding: true, style: { width: 32 } }}
          jumpProps={{ weight: 'shaded', color: 'primary', shape: 'brick', noPadding: true, style: { width: 32 } }}

</Playground>
